# Daylan Hoff - Portfolio

A modern, interactive portfolio website built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui. Features animated components, technical diagrams, and comprehensive skill showcases.

## üöÄ Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: AWS S3 + CloudFront
- **CI/CD**: GitHub Actions

- **Interactive Hero Section**: Animated gradient backgrounds and smooth transitions
- **Skills Visualization**: Animated progress bars showing proficiency levels

  - Technologies (.NET, React, Next.js, Node.js, Docker)
  - Cloud & DevOps (AWS, S3, EC2, GitHub Actions, CI/CD)
- **AWS Architecture Diagram**: Interactive visualization of deployment pipeline
- **Privacy-First Architecture**: C# and .NET security principles visualization
- **Expandable Experience Timeline**: Click to reveal detailed achievements

- **Fully Responsive**: Optimized for all screen sizes
